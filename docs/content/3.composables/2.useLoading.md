## useLoading

`useLoading` returns the function you can use to show loadings.

```ts
export type LoadingOptions = {
  text?: string;
  color?: string;
  background?: string;
  scale?: number;
  target?: string | HTMLElement;
};

// returns
export interface Loading {
  close: () => void;
}
```
